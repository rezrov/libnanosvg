libnanosvg
==========

## Purpose

This is https://github.com/memononen/nanosvg, repackaged as a shared library.

## Building (requires cmake)

```
$ cd src/build
$ cmake ..
$ make all

To install:

$ make install

To uninstall:

$ make uninstall
```

## Usage

Please refer to the documentation for nanosvg: https://github.com/memononen/nanosvg
